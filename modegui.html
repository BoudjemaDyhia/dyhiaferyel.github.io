Mode Graphique

Cette section abordera les détails de la version graphique en utilisant la bibliothèque Swing. Nous discuterons des composants d'interface, des interactions utilisateur, et des améliorations visuelles apportées à l'expérience utilisateur.
java -jar GUI.jar: Cette commande permet de lancer l'interface graphique à partir de console.

HomePage

La classe HomePage représente le point d'entrée visuel de notre application. Les utilisateurs peuvent ajouter un dossier à analyser en utilisant un bouton dédié. Une fois sélectionné, la classe MainPage prend le relais pour afficher l'arborescence complète du répertoire choisi.

MainPage

La méthode updateTreeModel() de la classe MainPage assure l'affichage de l'arborescence du répertoire sélectionné, incluant les fichiers Python à analyser. Des filtres intuitifs permettent aux utilisateurs de choisir le type d'analyse à appliquer aux fichiers, avec des résultats statistiques détaillés pour le répertoire analysé.

Result Page

La classe ResultPage offre une vue détaillée des résultats des analyses effectuées sur les fichiers Python du répertoire. Les informations fournies comprennent les annotations detype détectées, et d'autres analyses liées à la qualité du code.

Editeur de Code

Pour une manipulation plus approfondie, la classe PythonEditorFrame fournit un environnement dédié à l'édition des fichiers Python. Les utilisateurs peuvent effectuer des modifications directes grâce à cette interface, offrant une flexibilité accrue dans le processus de développement.
