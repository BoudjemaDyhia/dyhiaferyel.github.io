Mode Console

Cette section explore les fonctionnalités clés de notre mode console, détaillant les différentes options de ligne de commande, les actions disponibles et la manière dont notre application fournit des informations utiles aux utilisateurs. Nous mettons en lumière les principaux scénarios d'utilisation de mode console, démontrant comment elle facilite la gestion des fichiers Python, la détection d'annotations de type et l'évaluation de la qualité du code.

Les paramètres attendus en ligne de commande sont le nom du dossier ou du fichier à explorer, avec des options pour spécifier le type d'entrée et les actions souhaitées.

Exemples d'utilisations :

java -jar cli.jar -d ./: Liste tous les fichiers Python à partir du dossier spécifié par la classe Listfilepy.

java -jar cli.jar -f ./add.py --type: Analyser des fichiers Python pour détecter les annotations de type par la méthode runMypy().

java -jar cli.jar -f ./add.py --pydoc: Identifier les fonctions sans commentaires "PyDoc" dans les fichiers Python par la méthode checkpydoc().

java -jar cli.jar -f ./add.py –head: Vérifier la présence des commentaires d'en-tête par la méthode hascommentheader().

java -jar cli.jar -f ./add.py --comment: Ajouter les commentaires "PyDoc" dans les fichiers Python.

java -jar cli.jar -f ./add.py –subtf8: Ajouter les commentaires d'en-tête dans les fichiers Python.
